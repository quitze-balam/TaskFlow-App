# 📱 TaskFlow App

TaskFlow es una aplicación móvil Android desarrollada en Java que permite la gestión de actividades mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar), utilizando Firebase Firestore como base de datos en la nube.

---

## 🚀 Funcionalidades

- ➕ Registrar actividades con descripción y horas
- 📋 Visualizar lista de actividades
- 👀 Ver detalle de cada actividad
- ✏️ Editar actividades existentes
- 🗑️ Eliminar actividades
- 🔐 Validaciones de entrada de datos
- ☁️ Almacenamiento en Firebase Firestore

---

## 🧰 Tecnologías utilizadas

- Java (Android)
- Android Studio
- Firebase Firestore
- XML para interfaces
- Gradle

---

## 🔐 Seguridad implementada

- Validación de campos vacíos
- Validación de longitud de texto
- Validación de formato numérico
- Manejo de excepciones (try/catch)
- Reglas de seguridad en Firebase Firestore
- Confirmación de operaciones críticas

---

## 📊 Estructura del proyecto

- MainActivity → Lista de actividades
- RegistroActivity → Crear actividad
- DetalleActivity → Ver información
- EditarActivity → Modificar actividad

---

## 📸 Evidencias del sistema

- CRUD funcionando correctamente
- Datos almacenados en Firestore
- Validaciones en tiempo real
- Eliminación y actualización de registros

---

## ⚙️ Configuración del proyecto

1. Clonar repositorio
2. Abrir en Android Studio
3. Sincronizar Gradle
4. Conectar con Firebase
5. Ejecutar en dispositivo o emulador
